## 开始

  打开tomcat服务器，输入localhost:8080/WMS/login进入登录界面。然后开始注册账号或者直接登录。


## 登录日志记录

  登录过程中会生成日志文件，记录每个账号的登录信息。



## 后台管理

  客户管理
  供应商管理
  货物管理
  仓库管理
  
  
  
## 账号密码限制

  注册的密码存在特定规则，不符合要求不能注册。



## MD5加密

  对于用户账号的密码进行了MD5加密。



## 环境设定

  平台：eclipse 
  工具包版本：jdk1.7 
  数据库：Mysql 
  maven版本：maven3.1.0 
  服务器：tomcat
